# 🪶 SADDLEBAGS 🐴
*Pip & Biscuit's Delivery Co.* — a cozy pack-and-trot roguelite minigame for mobile.

**Play:** open `index.html` on any phone (or host on GitHub Pages — no build step, just static files).

**Code map:** see [`INDEX.md`](INDEX.md) for what lives in each file (`index.html`, `styles.css`, `data.js`, `engine.js`, `game.js`).

## How it plays
1. **Pack** — market crates drift by on a conveyor. Tap one and Pip the toucan snatches it; stow it in Biscuit's left or right saddlebag. Keep the weights balanced or items tumble off on the road! Biscuit starts each day a little low-spirited — feed him apples/carrots (tap his head) to extend his patience timer *and* lift his morale before you depart.
2. **Plan** — each day shows a delivery **Order** (big coins) and a partial **Route** forecast. Pack for the trouble ahead… what you pack *is* your build.
3. **Trot** — Biscuit journeys through 5 regions (Meadow, Woods, Windy Pass, Festival, Moonlight Bay). Encounters resolve with a packed item (a safe ~90% bet) or a riskier "Pip improvises" gamble. A fork mid-route offers a choice (pack a 🗺️ Map to reveal what's down each path).
4. **Spend** — coins buy horse coats, manes, toucan beaks, matching hats for both, and gear upgrades (bigger bags, comfy saddle, lucky horseshoe, spyglass, snack pouch). Any wardrobe purchase also nudges Biscuit's starting morale up a notch.

Progress is saved in `localStorage`.

## Deploy to GitHub Pages
Push this folder to a repo → Settings → Pages → deploy from branch → done.
